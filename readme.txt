Performance-focused visual WordPress page builder with all core features included.

== Description ==

Axiom WP Builder is a modern, extensible page builder plugin architecture with:
- template custom post types
- REST API for templates/settings/widgets
- admin builder shell
- extensibility hooks for widgets and integrations

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress Plugins menu.
3. Open `Axiom Builder` in admin menu.
